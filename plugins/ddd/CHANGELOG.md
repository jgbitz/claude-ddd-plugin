## [1.0.0] - 2025-10-20

### Added

- Initial version release
- Added `ddd-strategic-design` Skill
  - Input Diagnosis: analyze existing information and identify domain understanding gaps
  - Domain Exploration: discover business concepts and boundaries through structured questioning
  - Interview Planning: generate interview question sets for PMs, domain experts, and end users
  - Output Production: produce Bounded Context analysis, Context Map, and Ubiquitous Language
- Provided interview question templates
  - `pm-questions.md` - Product Manager interview framework
  - `expert-questions.md` - Domain expert interview framework
  - `user-questions.md` - End user interview framework
- Provided document output templates
  - `output-templates.md` - Standard DDD strategic document formats
